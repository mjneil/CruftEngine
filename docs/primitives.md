Built in data types in Javascript.



### Number
```javascript
let a = 1;
let b = 1.23456;
let c = a + b; //c = 2.23456
```

### Boolean
```javascript
let a = true;
let b = false;
let c = !b && a; //c = true
```

### String
```javascript
let a = "Cruft";
let b = "Engine";
let c = a + b; // c = "CruftEngine"
```

### Object
```javascript
let a = { 
	Cruft : true, 
	Engine :false 
}
```

### Function
```javascript
let add = (a, b) => {
	return a + b;
}
let c = add(6, 6);
```
